üéØ What is User Centered Design?

User Centered Design (UCD) is a design philosophy where end users' needs, wants, and limitations are given extensive attention at each stage of the design process.

UCD = Designing WITH users, FOR users

üß† Aspect 3: Ergonomics

Designing for human capabilities and limitations

üí™ Physical Ergonomics

  • Body posture: Neutral positioning
  • Repetitive movements: Minimize strain
  • Workspace layout: Optimal arrangement
  • Comfort: Reduce physical fatigue
Example: Ergonomic mouse with curved design reduces wrist strain

üß† Cognitive Ergonomics

  • Mental workload: Keep it manageable
  • Decision-making: Clear choices
  • Memory: Don't overload (7¬±2 items)
  • Attention: Minimize distractions
Example: Chunking phone numbers (123-456-7890) aids memory

üé® Aspect 4: Aesthetics

Visual appeal and emotional design

Key Aesthetic Elements

  • Visual Hierarchy: Guide eye to important elements
  • Color Theory: Harmonious color palettes
  • Typography: Readable and appropriate fonts
  • Whitespace: Proper breathing room
  • Consistency: Unified visual language
  • Balance: Visual equilibrium

üí° Aesthetic-Usability Effect

Users often perceive aesthetically pleasing designs as more usable, even when functionality is identical. Beautiful designs create positive emotional responses!

üéØ Key Takeaways

‚úÖ Remember These Points

  • UCD is iterative - Design, test, refine, repeat
  • Always focus on users - Their needs come first
  • Four aspects matter equally - Don't neglect any
  • Test with real users - Assumptions can be wrong
  • Aesthetics affect perception - Beautiful = Usable
  • Balance is key - All aspects must work together
